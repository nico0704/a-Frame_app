Multimediasysteme 3 - Setup

1. Password & ggf. User (default root) für MySql in app.js eintragen (Line 13 - 15)

-> Wenn aframe Database automatisch aufgesetzt werden soll, einfach "node app.js" starten und localhost:8080 aufrufen.

-> Wenn aframe Database schon vorhanden ist:

1. In app.js line 9 "db_ready" auf true setzen.

2. In app.js line 16 Namen der DB eintragen, wenn diese nicht "aframe" heißt.

3. In app.js line 27 auskommentieren.

4. node app.js starten und localhost:8080 aufrufen.
